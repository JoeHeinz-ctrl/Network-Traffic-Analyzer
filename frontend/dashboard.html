📐 Mathematical Analysis & Course Outcomes

Detailed mathematical foundations and formulations applied to network traffic analysis

CO1

Partial Differential Equations (PDE)

Application: Gaussian filtering for noise reduction using heat diffusion equations

Heat Equation (Diffusion PDE):
∂u/∂t = α∇²u
where u(x,t) is traffic intensity, α is diffusion coefficient
Gaussian Kernel:
G(x,y,σ) = (1/2πσ²)e^(-(x²+y²)/2σ²)
Applied with σ = 1.5 for optimal smoothing
Smoothing Method: Gaussian Convolution
Noise Reduction: ~40-60%
CO2

Linear Transformations

Application: Feature normalization and scaling for uniform data representation

Min-Max Normalization:
x' = (x - x_min) / (x_max - x_min)
Maps features to [0, 1] range
Z-Score Standardization:
z = (x - μ) / σ
Centers data with mean=0, std=1
Features Normalized: packet_size, protocol, timestamp
Transformation Matrix: T: ℝⁿ → ℝⁿ (linear map)

Orthonormal Bases & Inner Product

Application: Principal Component Analysis for dimensionality reduction

Covariance Matrix:
Σ = (1/n)X^T X
Captures feature correlations
Eigenvalue Decomposition:
Σv = λv
Eigenvectors form orthonormal basis
PCA Projection:
Z = XW, where W = [v₁, v₂]
Projects to 2D principal components
